# Library

- The Library where you can add book information and remove it too

- You can add Book tilte , author , number of Pages and read Status of the Book

- This project focuses on showing constructor and prototype logic of javascript

### How to add book-

1. Click on "Add Book" Button on top-right
2. Enter book info
3. Click Add Book
4. Use "Remove book🗑️" to remove a book from library
5. Use "Read"/"Not Read" buttons to change the read status
